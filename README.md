## 概要
このアプリは、外出前にテレビやエアコンなど電源を切らなければならないものや、扉の施錠など、確認をしなければいけないものをリスト化し、順番に確認するものです。設定画面で確認項目を入力し、「確認する」ボタンを押すと、順番に設定で入力した確認項目が表示されるようになっています。
